# Use Case: Cập nhật Hồ sơ Chuyên môn

- **Actor**: Developer[cite: 1].
- **Pre-condition**: Developer đã đăng nhập thành công.
- **Flow**:
  1. Dev chọn mục "Cập nhật hồ sơ".
  2. Dev tải lên file CV mới (PDF/DOC).
  3. Dev cập nhật danh sách kỹ năng và thành tích mới.
  4. Hệ thống lưu tạm và gửi thông báo cho Leader/HR phê duyệt.
- **Post-condition**: Hồ sơ được cập nhật sau khi phê duyệt.
